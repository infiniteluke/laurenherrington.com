## Setup
* `yarn`
* `touch .env`
* Add keys `CONTENTFUL_SPACE_ID` and `CONTENTFUL_ACCESS_TOKEN` to `.env`
* `yarn develop`

## Entering Content
* Login at https://app.contentful.com/spaces/ and choose the "space" for this site
